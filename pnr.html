What is PNR?

A PNR number, which stands for Passenger Name Record, is a unique 10-digit alphanumeric code assigned to each passenger's booking when they book a ticket for Railways,airways or any other Transport This number acts as a key to access and retrieve details about the booked ticket, including passenger details, travel information, and booking status.
